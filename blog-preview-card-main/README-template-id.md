### Tangkapan layar

![](./screenshot.jpg)

Tambahkan tangkapan layar solusi Anda. Cara termudah untuk melakukannya adalah menggunakan Firefox untuk melihat proyek Anda, klik kanan halaman dan pilih "Ambil Tangkapan Layar". Anda dapat memilih tangkapan layar penuh tinggi atau yang dipotong berdasarkan panjang halaman. Jika sangat panjang, sebaiknya potong saja.

Alternatifnya, Anda dapat menggunakan alat seperti [FireShot](https://getfireshot.com/) untuk mengambil tangkapan layar. FireShot memiliki opsi gratis, jadi Anda tidak perlu membelinya.

Kemudian potong/optimalkan/edit gambar Anda sesuka Anda, tambahkan ke proyek Anda, dan perbarui jalur file di gambar di atas.

**Catatan: Hapus catatan ini dan paragraf di atas saat Anda menambahkan tangkapan layar. Jika Anda lebih suka tidak menambahkan tangkapan layar, silakan hapus seluruh bagian ini.**
